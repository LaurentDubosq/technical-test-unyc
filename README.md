## Project Setup

```sh
git clone https://github.com/LaurentDubosq/technical-test-unyc.git
```

```sh
cd technical-test-unyc
```

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
